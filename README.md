# The Adaptive Brain

An AI-powered exam preparation platform that adapts to your learning patterns. Built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Charts:** Recharts
- **Icons:** Lucide React
